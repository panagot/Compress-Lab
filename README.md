# Compress Lab

Open developer-education prototype for **Solana ZK Compression & Privacy DX** — recipes, local workbench, architecture, grant brief.

Application tracks:
- Solana Foundation — Developer Education & Ecosystem
- Superteam Balkan grants (regional workshop distribution)

## Run

```bash
cd solana/zk-lab
npm install
npm run dev
```
